# 📝 MeetBrief AI: Native Full-Stack Summarizer

A lightweight, dependency-free full-stack web application that takes conversational meeting transcripts and dynamically structures automated summaries and action items.

## 🚀 Features
- **Native Java Backend:** Built purely using Java's core `HttpServer`.
- **Modern Minimal UI:** A premium, single-page client interface with a lavender aesthetic.
- **One-Click Clipboard:** Includes a "Copy Full Brief" feature for instant data usage.

## 💻 How to Run
1. Run the `MeetBrief.java` server.
2. Open `index.html` in your browser.
